# Spotsylvania RFC Brand Guide

## Core identity

**Official name:** Spotsylvania Rugby Football Club  
**Primary public name:** Spotsylvania RFC  
**Plain-language public name:** Spotsylvania Rugby  
**Youth program name:** Spotsylvania Youth Rugby  
**Tag rugby program name:** Spotsylvania Tag Rugby  
**Optional youth/social shorthand:** Spotsy Rugby  
**Practical monogram:** SRFC  
**Secondary graphic code:** SPLV  
**Founded:** Est. 2026  
**Motto:** Unity · Courage · Service

## Brand statement

Spotsylvania RFC is a traditional rugby football club in formation, built on unity, courage, service, local pride, and lifelong fellowship.

## Brand personality

The club should feel:

- Traditional
- Local
- Disciplined
- Welcoming
- Rugged
- Virtuous
- Formational
- Family-friendly
- Heritage-minded
- Community-rooted

Avoid anything that feels:

- Gimmicky
- Corporate
- Flashy
- Cartoonish
- Esports-like
- Aggressive for its own sake
- Overtly religious

The Catholic-adjacent tone should come through virtue language, tradition, service, formation, discipline, and fellowship, not through explicit religious symbols.

## Color palette

### Primary colors

| Color             |       Hex | Use                                                    |
| ----------------- | --------: | ------------------------------------------------------ |
| Deep Navy         | `#0D1F2D` | Primary background, formal identity, nav, dark apparel |
| Forest Green      | `#1A3A2A` | Rugby field, secondary panels, alternate kit           |
| Antique Gold      | `#C8963E` | Crest linework, buttons, trim, ceremonial details      |
| Cream / Parchment | `#F5EFE3` | Warm backgrounds, text on dark fields                  |

### Supporting colors

| Color       |       Hex | Use                   |
| ----------- | --------: | --------------------- |
| Warm Light  | `#EDE4D3` | Secondary panels      |
| Dark Text   | `#1C1208` | Body copy             |
| Muted Brown | `#6B5E4A` | Captions, helper text |

### Color hierarchy

Navy should dominate. Green should support. Gold should accent. Cream should give warmth and contrast.

Avoid using gold as a large background color. It works best as trim, linework, button color, borders, and ceremonial accent.

## Typography

| Use                         | Font              |
| --------------------------- | ----------------- |
| Display / heritage headings | Playfair Display  |
| Body copy                   | Libre Baskerville |
| Labels / buttons / nav      | Oswald            |

Use Playfair Display for major headings, formal print pieces, hero text, and ceremonial language. Use Libre Baskerville for paragraphs and longer copy. Use Oswald for navigation, labels, buttons, badges, and short uppercase text.

## Logo system

### Primary crest

Use for:

- Website hero
- Posters
- Social graphics
- Sponsor decks
- Flyers
- Print materials
- Banners
- Email headers

Preferred file formats:

- SVG if available
- Transparent PNG for web
- PDF, EPS, or AI for print/vendor use

### Kit crest

Use for:

- Jerseys
- Shorts
- Polos
- Hats
- Training gear
- Embroidery
- Small apparel applications

The kit crest should be simplified, without motto, date, or tiny acorns. It should remain readable at small chest-crest size.

### Small mark

Create an **SRFC** small mark for favicon, social avatar, hats, shorts, small embroidery, and watermarks.

Create an optional **SPLV** heritage code mark for apparel details, campaign graphics, inside collar marks, and poster accents.

## Secondary heritage layer

**Forged in Spotsylvania** may be used as a secondary campaign or apparel phrase.

It should support themes of formation, durability, craft, discipline, and local rootedness.

Use foundry / iron / forged language carefully. Do not make cannons a core club symbol.

## Language and voice

Use words like:

- Unity
- Courage
- Service
- Fellowship
- Tradition
- Formation
- Discipline
- Commitment
- Sportsmanship
- Local pride
- Founding list
- Earn the badge
- Build the club
- Put Your Hand Up
- In formation
- Forged in Spotsylvania

Avoid words like:

- Elite
- Dominate
- Crush
- Warrior
- Lifestyle brand
- Disrupt
- Exclusive
- Premium experience

Rugby can be tough without sounding performatively aggressive.

## Founding campaign phrase

Use **Put Your Hand Up** as a broad rallying phrase for parents, players, coaches, volunteers, sponsors, teachers, and supporters.

Pair it with clear CTAs such as **Join the Founding List**.

## Apparel guidance

### Home kit

- Deep navy base
- Antique gold trim
- Cream numbers
- Optional forest green accent

### Alternate kit

- Forest green base
- Antique gold trim
- Cream numbers
- Navy details

### Training / social kit

- Cream shirt
- Navy or green crest
- Gold accent

### Motto placement

The motto does not need to appear inside the kit crest. Better options:

- Inside collar: Unity · Courage · Service
- Sleeve: Est. 2026
- Back neck: Spotsylvania RFC or SRFC
- Founding shirt: Forged in Spotsylvania

## Brand rules of thumb

When deciding whether something fits Spotsylvania RFC, ask:

- Does it feel traditional?
- Does it feel local?
- Does it feel durable?
- Does it welcome families and beginners?
- Does it still feel like rugby?
- Would it look good on a navy jersey?
- Would it still make sense ten years from now?

When in doubt, choose the simpler, more traditional option.
